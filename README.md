# АЛАРОС

Сайт Ассоциации ландшафтных архитекторов России: [alaros.ru](https://alaros.ru)

Публичный сайт ассоциации с новостями, мероприятиями, каталогом проектов, партнерскими разделами и фестивальной программой. Контент управляется через настроенную CMS Strapi.

## О проекте

На сайте реализованы основные разделы организации:

- главная страница с афишей событий, новостями и партнерами;
- каталог проектов с фильтрацией и детальными модальными карточками;
- раздел новостей и отдельные страницы статей;
- страницы фестиваля, программы, жюри, документов, заявок и результатов конкурса;
- разделы об ассоциации, партнерах и контактах;
- интерактивные формы с защитой reCAPTCHA;
- карта, слайдеры, анимации переходов и плавный скролл;
- интеграция с CMS Strapi для редактирования контента без участия разработчика.

## Моя роль

- Разработка frontend-части на Next.js и React.
- Настройка маршрутизации, серверных страниц и API routes.
- Настройка Strapi CMS для управления контентом сайта.
- Проектирование структуры данных, коллекций и одиночных типов в Strapi.
- Интеграция frontend-приложения с Strapi API.
- Реализация адаптивной верстки и UI-компонентов.
- Подключение форм, reCAPTCHA и отправки писем.
- Работа с производительностью, кэшированием и SEO-метаданными.
- Координация отдельных задач разработки: постановка задач, ревью результата и контроль качества реализации.
- Подготовка проекта к production-сборке.
- Деплой на сервер с Nginx.

## Особенности реализации

- Используется App Router и серверные компоненты Next.js.
- CMS Strapi настроена под структуру сайта: страницы, новости, проекты, партнеры, документы и фестивальные разделы управляются из админ-панели.
- Контент загружается из Strapi через typed fetch-утилиты.
- Для данных применяется `unstable_cache` и тегированная ревалидация.
- Контактные формы обрабатываются через Next.js API routes.
- Для защиты форм подключена серверная проверка Google reCAPTCHA.
- Реализованы кастомные слайдеры, модальные окна, пагинация, карта и анимированные переходы.

## Локальный запуск

```bash
npm install
npm run dev
```

После запуска сайт будет доступен по адресу:

```bash
http://localhost:3000
```

## Переменные окружения

Для полноценной работы проекта нужны переменные окружения:

```env
API_URL=
API_TOKEN=
NEXT_PUBLIC_IMAGES_URL=
RECAPTCHA_SECRET_KEY=
EMAIL_USER=
EMAIL_PASS=
EMAIL_TO=
```
